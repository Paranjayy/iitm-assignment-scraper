# New Feature Ideas

Brainstorm of features to add to the IITM Portal Extension.

---

## 🔥 Priority (Now/Next/Later)

### Now (High Impact, Easy)
1. **Auto-extract from dashboard** — Scrape active courses automatically without manual navigation
2. **Duplicate test case cleanup** — Deduplicate cases by input+expected fingerprint before export
3. **OPPE-specific scraper** — Dedicated extractor for OPPE portal format

### Next (High Impact, Medium Effort)
4. **Scrape Quality Score** — Rate each export on frontmatter completeness, topic presence, test purity
5. **Frontmatter Auto-Fixer** — Auto-patch missing topic, normalize breadcrumb, enforce title matching
6. **Topic Inference Engine** — If topic missing, infer from question text and tab labels
7. **Export Diff Mode** — When re-exporting, show what changed: tests, statement, solution
8. **Bulk Run Report** — Summary markdown: captured, skipped, locked tabs, retries, mismatch reasons
9. **One-Click Debug Bundle** — Sanitized bug bundle with DOM snapshots and scrape logs

### Later (Medium Impact, Higher Effort)
10. **AI Study Pack Export** — Per assignment: key concepts, common mistakes, revision questions
11. **Assignment Difficulty Heuristic** — Estimate difficulty from code complexity + test depth
12. **Course Timeline View** — Weekly progress timeline from scraped data
13. **Resource Link Health Checker** — Validate drive/docs/pdf links, flag dead links
14. **Transcript Chunk Indexing** — Split transcripts into semantic chunks with timestamps
15. **Personalization Layer** — User chooses strict/speed/safe mode for bulk scraping
16. **Regression Fixture Suite** — Save sample HTML fixtures for extraction tests
17. **Locked-Tab Rehydration** — Placeholder files that auto-update when lock lifts
18. **OPPE Boundary Validator** — Detect harness suffix and enforce separator markers

---

## 💡 Creative Ideas

- **GPA Calculator** — Auto-compute CGPA from all exported score data
- **Assignment Calendar** — Export deadlines to Google Calendar / Apple Calendar
- **Study Group Sharing** — Share notes/exports with classmates via link
- **Performance Dashboard** — Visual charts of scores over time (fetch from Score Checker)
- **Smart Notifications** — "Assignment due in 2 hours" alerts
- **Code Snippet Library** — Save frequently used code snippets for quick paste into IDE
- **Plagiarism Checker** — Compare your solution against known patterns
- **Offline Mode** — Cache full course content for offline viewing
